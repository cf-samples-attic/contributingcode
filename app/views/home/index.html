Bootcamp

The web applications built for the contributing code sprint are to be deployed on cloudfoundry.(PASS). Bootcamp is the ideal place to come and learn about the latest developments in web applications and to deploy your awesome web applications on cloudfoundry. Please check the schedule for more details.

Rules

  • Its a two week code sprint
  • Themes will be told on the day on bootcamp
  • Each team will given a git repo to push the code
  • Teams can start coding once the git repo is given
  • Apps are to be deployed on cloudfoundry
  • Teams showing up on July 28th for the last 12 hr hacking are only eligible for prizes
  • Each team is required to have a maximum of 5 members
